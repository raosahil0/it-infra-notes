# 🔐 Access Control Scenarios

---

## 📌 Purpose

Access control ensures **only authorized users** can access systems and resources.

---

## ⚙️ Common Scenarios

### 1. User Unable to Access Application

#### Causes
- Missing permissions  
- Incorrect role assignment  
- Access policy restriction  

#### Troubleshooting
- Verify user role  
- Check access policy  
- Confirm application permissions  

#### Fix
- Assign correct role  
- Update access policy  

---

### 2. VPN Connected but No Access

#### Causes
- User not mapped to correct network  
- Firewall blocking internal traffic  

#### Troubleshooting
- Check VPN user group  
- Verify firewall rules  

#### Fix
- Assign correct group  
- Allow internal access in firewall  

---

### 3. Unauthorized Access Attempt

#### Causes
- Invalid credentials  
- Security policy block  

#### Troubleshooting
- Check authentication logs  
- Verify login attempts  

#### Fix
- Reset credentials  
- Enforce policy  

---

## 🔥 Real-World Insight

Access issues are usually:
- Permission-related  
- Policy-related  

Not system failures.
