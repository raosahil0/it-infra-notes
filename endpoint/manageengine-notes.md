# 🛠️ ManageEngine Endpoint Central Notes

---

## 📌 Overview

ManageEngine Endpoint Central is used for:
- Device management  
- Patch deployment  
- Software installation  
- Remote troubleshooting  

---

## 🔧 Key Components

- Agent (installed on endpoints)
- Central server
- Repository (for patches/software)

---

## ⚙️ Common Tasks

### 1. Agent Installation

- Install agent on endpoint  
- Ensure device connects to server  

---

### 2. Patch Management

- Scan for missing patches  
- Approve patches  
- Deploy to target devices  

---

### 3. Remote Troubleshooting

- Remote desktop access  
- Command execution  
- System diagnostics  

---

### 4. Policy Management

- Assign policies to device groups  
- Enforce security configurations  

---

## ⚠️ Common Issues

### Agent Not Reporting
- Cause: Service stopped / network issue  
- Fix: Restart agent / check connectivity  

---

### Patch Not Applied
- Cause: Device offline / failed download  
- Fix: Re-deploy patch  

---

### Device Not Visible
- Cause: Agent not installed / not synced  
- Fix: Reinstall agent  

---

## 🔥 Real-World Insight

- Endpoint tools depend heavily on **agent health**  
- If agent fails → entire management fails  
- Always check:
  - Agent status  
  - Connectivity  
  - Sync status
