# 🔐 VPN Basics

---

## 📌 What is VPN?

A VPN (Virtual Private Network) creates a **secure encrypted tunnel** between a user device and a private network.

---

## ⚙️ Why VPN is Used

- Secure remote access to internal systems  
- Protect data over public networks  
- Access internal applications  

---

## 🔄 How VPN Works (Simple Flow)

1. User connects to VPN client  
2. Authentication happens (username/password)  
3. Encrypted tunnel is established  
4. Traffic is routed through VPN server  
5. User gets access to internal resources  

---

## ⚠️ Key Components

- VPN Client (user device)
- VPN Server (firewall)
- Authentication system
- Encryption protocol

---

## 🔥 Real-World Insight

- VPN depends heavily on:
  - network stability  
  - firewall capacity  
  - user authentication  

If any fails → VPN issues occur
